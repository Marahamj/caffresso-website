# ☕ Caffresso Website

A modern and responsive coffee shop website built using HTML, CSS, and JavaScript.

## 📌 Overview

Caffresso is a simple front-end project that presents a coffee shop with a clean and elegant design. The website includes multiple pages and interactive features to improve the user experience.

## ✨ Features

- Responsive design
- Modern navigation bar
- Hero section
- Coffee menu with category tabs
- Best-selling products
- Customer testimonials
- About Us section
- Contact form
- Footer with social media links
- Hover effects and smooth transitions

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome
